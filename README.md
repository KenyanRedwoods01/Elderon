## 🚀 Quick Start

### Prerequisites
- **Node.js** 18.0 or higher
- **npm** 9.0 or higher  
- **Git**
- **Docker** (optional, for containerized deployment)

### Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/KenyanRedwoods01/Elderon.git
cd Elderon

# 2. Install dependencies
npm install

# 3. Set up environment configuration
cp .env.example .env.local

# 4. Configure your environment variables
# Edit .env.local with your specific settings

# 5. Start the development server
npm run dev

# 6. Open your browser and navigate to
# http://localhost:3000
```

### Docker Deployment

```bash
# Quick start with Docker Compose
docker-compose up -d

# Or build custom images
docker build -t elderon-platform .
docker run -p 3000:3000 -p 5432:5432 elderon-platform
```

### Kubernetes Deployment

```bash
# Deploy to Kubernetes cluster
kubectl apply -f infrastructure/kubernetes/

# Monitor deployment
kubectl get pods -n elderon
kubectl get services -n elderon
```
